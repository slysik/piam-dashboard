# Video Clips for PIAM Demo

Place sample video clips here for the Video Evidence feature.

## Expected Files

- `badge-access-lobby.mp4` - Person badging into main lobby
- `badge-access-server-room.mp4` - Person accessing secure server room
- `badge-denied-entry.mp4` - Access denied scenario

## Free Stock Video Sources

Download badge access / door entry footage from:

1. **Pexels** - https://www.pexels.com/search/videos/badge/
2. **Pexels Door Videos** - https://www.pexels.com/search/videos/door/
3. **Pixabay Badge** - https://pixabay.com/videos/search/badge/
4. **Pixabay Security** - https://pixabay.com/videos/search/security%20door/

Search terms that work well:
- "badge access"
- "card reader"
- "security door"
- "building entry"
- "office entrance"

## Video Specifications

- Format: MP4 (H.264)
- Duration: 5-15 seconds
- Resolution: 720p or 1080p
- Aspect ratio: 16:9 preferred

## Usage

Once you add videos here, they will automatically appear in the Video Evidence modal when clicking the video icon in the Real-Time Risk panel.
